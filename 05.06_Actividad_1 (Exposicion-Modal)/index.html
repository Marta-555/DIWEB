COMPONENTE MODAL

Características y estructura

  • El complemento Modal de Bootstrap se suele utilizar para agregar cuadros de diálogo, notificaciones de usuario o contenido adicional.
  • Al activar este componente se crea una ventana que se posiciona sobre el resto de elementos, oscureciendo y deshabilitando el resto de la página.
  • data-bs-toggle: indicamos que vamos a abrir un modal
  • data-bs-target: especificamos el ID del modal que vamos a abrir
